# itfxq_travel
旅游网系统 使用 vue全家桶 + springboot+ ssm 打造，不管是旅游网的前端页面 还是旅游网后台管理页面都是vue打造，界面不借用任何模板，全手工写出来。同时也是前后端分离项目。是您学习，开发， 还有使用 的不错选择。
